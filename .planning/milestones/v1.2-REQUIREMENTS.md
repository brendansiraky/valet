## Multi-Provider Support (MPROV)

- [x] **MPROV-01**: System has abstraction layer for AI providers
- [x] **MPROV-02**: Anthropic provider implements abstraction layer
- [x] **MPROV-03**: OpenAI provider implements abstraction layer
- [x] **MPROV-04**: User can add API key per provider in settings
- [x] **MPROV-05**: Provider gracefully handles missing capabilities (best effort parity)

## Model Selection (MODL)

- [x] **MODL-01**: User sees flat model dropdown grouped by provider
- [x] **MODL-02**: User can mix models from different providers in same pipeline
- [x] **MODL-03**: Model dropdown shows only providers with configured API keys
- [x] **MODL-04**: Default model respects user's global preference per provider

## Agent Updates (AGNT)

- [x] **AGNT-11**: Agent-pipeline relationship is live link (not snapshot)
- [x] **AGNT-12**: Orphan handler for deleted agents in pipelines
- [x] **AGNT-13**: Agent can use any available model (cross-provider)

## Artifact Storage (ARTF)

- [x] **ARTF-01**: Pipeline outputs stored in database with metadata
- [x] **ARTF-02**: User can view past pipeline run outputs
- [x] **ARTF-03**: Artifacts stored as structured JSONB (not plain text)
- [x] **ARTF-04**: Artifact metadata includes run date, pipeline, cost, model
